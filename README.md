# 🧠 Санитар Дурдома — Семейный VPN & MTProto Proxy Bot

Telegram-бот для управления VPN конфигами и персональными MTProto-прокси для семейного/корпоративного использования.

![Python](https://img.shields.io/badge/Python-3.11+-blue.svg)
![aiogram](https://img.shields.io/badge/aiogram-3.x-green.svg)
![License](https://img.shields.io/badge/License-MIT-yellow.svg)

## ✨ Возможности

### Для пользователей
- 🔐 **VPN конфиги** — получение, скачивание, запрос новых
- 🛰 **Персональные прокси** — выписка с названиями, подключение в 1 клик
- ⏰ **Индикаторы срока** — визуальные метки /⚠️/✅ для истекающих ключей
- 📱 **Кнопочное меню** — без ввода команд, всё через inline-кнопки
-  **Уведомления** — автоматические напоминания об истечении срока

### Для администратора
- 📊 **Статистика** — топ-10 активных пользователей
- 🔍 **Проверка сроков** — VPN и прокси в реальном времени
-  **Управление конфигами** — просмотр, удаление, очистка
- 📦 **Бэкапы** — ручные + автоматические (cron в 03:00)
- 📋 **Аудит действий** — логирование всех админских операций

### Безопасность
- 🔒 Конфиденциальные данные только в ЛС (проверка `chat.type`)
- 🛡 Rate Limiting — защита от спама
- 📁 Path Traversal защита — валидация путей
-  Токен бота в `.env`, права `chmod 600`
-  Аудит действий администратора
